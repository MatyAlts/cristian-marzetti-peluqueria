# Estructura de Imágenes

Esta carpeta contiene todas las imágenes del proyecto.

## Carpetas:

### `/hero`
Imágenes para el banner principal de la página de inicio
- Tamaño recomendado: 1920x1080px o superior
- Formato: JPG, PNG o WebP

### `/nosotros`
Imágenes para la sección "Nosotros"
- Fotos del salón, ambiente, detalles
- Tamaño recomendado: 800x600px o superior

### `/equipo`
Fotos del equipo de trabajo
- Fotos individuales de cada miembro
- Tamaño recomendado: 400x600px (vertical)
- Formato: JPG o PNG con fondo

### `/academia`
Imágenes para la sección de cursos y academia
- Fotos de clases, estudiantes, certificados
- Tamaño recomendado: 600x400px o superior

### `/productos`
Imágenes de productos
- Fotos de productos individuales
- Tamaño recomendado: 400x400px (cuadrado)
- Fondo blanco o transparente preferiblemente

## Notas:
- Optimiza las imágenes antes de subirlas (usa herramientas como TinyPNG)
- Usa nombres descriptivos (ej: `hero-salon.jpg`, `producto-shampoo.jpg`)
- Mantén el tamaño de archivo bajo para mejor rendimiento web
